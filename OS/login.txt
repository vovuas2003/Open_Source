Дистанционно с винды

putty - консоль
hostname: ДАННЫЕ УДАЛЕНЫ
port: ДАННЫЕ УДАЛЕНЫ
слева translation кодировка utf-8
слева session ввести имя (н-р, MIPT) в saved session
нажать save, потом можно load, чтобы не вбивать всё это опять
снизу справа open (при первом запуске accept)
login: ДАННЫЕ УДАЛЕНЫ
password: ДАННЫЕ УДАЛЕНЫ

WinSCP - файловый менеджер (как проводник в винде, но между 2 компами)
всё аналогично: hostname, port, save, login, первый раз yes

для выхода из putty и WinSCP просто закрыть окно

===========================================================================

С компа в учебном классе

для консольного доступа в терминале ввести команду
ssh -X login@hostname
password: ДАННЫЕ УДАЛЕНЫ
login: ДАННЫЕ УДАЛЕНЫ
для выхода из ssh ввести logout (вроде можно ещё ctrl+d)

файловый менеджер (на момент написания этого блокнота не проверял ещё)
sftp login@hostname

===========================================================================

Дистанционно с линукса

ssh -p port_number_here hostname
sftp -P port_number_here hostname (или -P port_number_here переместить в конец)
